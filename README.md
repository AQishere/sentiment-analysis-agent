# Sentiment Analysis Agent

An automated machine learning agent designed to classify text inputs into positive, negative, or neutral sentiment distributions.

## Key Features
* **Interactive Dashboard:** Deploys a Streamlit user interface for real-time review evaluation and metric visualization.
* **Automated Data Processing:** Standardizes and cleans raw text strings, applying tokenization and stop-word filtering for pipeline intake.
* **Pattern Classification:** Utilizes text-processing algorithms and classification models to accurately map text signatures against distinct sentiment scales.

## Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Libraries:** Scikit-Learn, Pandas, NumPy, NLTK

## How to Run
Launch the web application directly from your local terminal:
```bash
streamlit run app.py
